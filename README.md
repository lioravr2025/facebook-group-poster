# Facebook Group Auto-Poster

> **v1.2.0 — גרסה יציבה ועובדת תקין ✅**

תוסף Chrome לפרסום אוטומטי בקבוצות פייסבוק — פוסטים ותגובות — עם ממשק RTL בעברית.

---

## תכונות

- **פרסום פוסט חדש** בקבוצות מרובות ברצף אוטומטי
- **פרסום תגובה** בפוסט הראשון בכל קבוצה
- **כותרת** בשורה נפרדת + **גוף פוסט** עם שמירת רווחים מדויקת
- **קישור** עם המתנה לטעינת תצוגה מקדימה לפני הפרסום
- **השהיה אקראית** 110–130 שניות בין קבוצה לקבוצה (נגד חסימות)
- **ייבוא רשימה** — הדבקת עשרות קישורים בבת אחת
- **לוג ריצה** מפורט עם העתקה וניקוי
- אייקון בסרגל הכלים של Chrome

---

## התקנה

1. הורד את הקבצים (או שכפל את הריפו)
2. פתח Chrome → `chrome://extensions`
3. הפעל **Developer mode** (פינה ימנית עליונה)
4. לחץ **Load unpacked** ובחר את תיקיית הפרויקט
5. האייקון יופיע בסרגל הכלים

---

## שימוש

1. לחץ על אייקון התוסף בסרגל הכלים
2. מלא את שדות התוכן:
   - **כותרת הפוסט** — תופיע בשורה ראשונה מובלטת
   - **גוף הפוסט** — רווחי שורה נשמרים כמות שהם
   - **קישור** (אופציונלי) — יופיע שתי שורות ריקות מתחת לתוכן
3. הוסף קבוצות — אחת אחת או לחץ **📋 ייבוא** להדבקת רשימה שלמה
4. לחץ **התחל פרסום פוסט** או **התחל פרסום תגובה**
5. התוסף פותח כל קבוצה בכרטיסייה, ממתין לטעינה, מפרסם, סוגר ועובר לבאה
6. לוג הריצה מוצג בתחתית הפופאפ

## קבצים

| קובץ | תפקיד |
|------|--------|
| `manifest.json` | הגדרות התוסף (Manifest V3) |
| `popup.html` | ממשק המשתמש |
| `popup.js` | לוגיקת UI, ייבוא קבוצות, לוג |
| `background.js` | תור עבודה, פתיחת לשוניות, תזמון |
| `content.js` | הזרקת DOM לפייסבוק וביצוע הפרסום |
| `icons/` | אייקונים 16 / 48 / 128 px |
