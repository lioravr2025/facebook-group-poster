# Facebook Group Auto-Poster

> **v2.0.0 — גרסה יציבה ועובדת תקין ✅**

תוסף Chrome לפרסום אוטומטי בקבוצות פייסבוק — פוסטים ותגובות — עם ממשק RTL בעברית מלא.

---

## תכונות

### פרסום
- **פרסום פוסט חדש** בקבוצות מרובות ברצף אוטומטי
- **פרסום תגובה** בפוסט הראשון בכל קבוצה
- **כותרת** בשורה נפרדת + **גוף פוסט** עם שמירת רווחים מדויקת
- **קישור** עם המתנה לטעינת תצוגה מקדימה לפני הפרסום

### Spintax
- תמיכה ב-Spintax לגיוון אוטומטי: `{אפשרות1|אפשרות2|אפשרות3}`
- כל קבוצה מקבלת וריאציה שונה — נגד זיהוי ספאם
- עובד גם בכותרת וגם בגוף הפוסט

### ניהול קבוצות
- **רשימות קבוצות** — שמירת קבוצות בשם ושימוש חוזר
- **ייבוא מרובה** — הדבקת עשרות קישורים בבת אחת
- ניהול ועריכת כל רשימה בנפרד

### תזמון
- **קמפיינים מתוזמנים** — הגדרת תאריך ושעה לפרסום עתידי
- הקמפיין מופעל אוטומטית גם כשהדפדפן סגור (chrome.alarms)
- היסטוריית קמפיינים עם סטטוס: ממתין / רץ / הצליח / נכשל

### השהיות ותצוגה
- **השהיה מותאמת** — מינימום ומקסימום שניות בין קבוצות (ברירת מחדל: 110–130 שנ׳)
- **דשבורד התקדמות** — פס הת��דמות, ספירה לאחור לפרסום הבא, סטטיסטיקות
- **לוג ריצה** מפורט עם צבעים (הצלחה / שגיאה / אזהרה), העתקה וניקוי
- אייקון בסרגל הכלים של Chrome

---

## התקנה

1. הורד את הקבצים (או שכפל את הריפו)
2. פתח Chrome → `chrome://extensions`
3. הפעל **Developer mode** (פינה ימנית עליונה)
4. לחץ **Load unpacked** ובחר את תיקיית הפרויקט
5. האייקון יופיע בסרגל הכלים

---

## שימוש

### פרסום ידני
1. לחץ על אייקון התוסף → לשונית **ראשי**
2. מלא את שדות התוכן (כותרת, גוף, קישור)
3. בחר **רשימת קבוצות** מהתפריט הנפתח
4. הגדר השהיה מינימלית ומקסימלית
5. לחץ **פרסם פוסט** או **פרסם תגובה**

### ניהול קבוצות
1. לשונית **קבוצות** → **+ רשימה חדשה**
2. שם הרשימה → הוסף קבוצות אחת-אחת או **📋 ייבוא רשימה**
3. הדבק URLs (שורה לכל קבוצה) ולחץ ייבא

### תזמון קמפיין
1. לשונית **תזמון** → מלא פרטי קמפיין
2. בחר תאריך, שעה, רשימת קבוצות וסוג פרסום
3. לחץ **תזמן קמפיין** — הקמפיין יופעל אוטומטית

### דוחות
- לשונית **דוחות** — לוג הריצה האחרונה עם כל הצעדים שבוצעו

|------|--------|
