# Facebook Group Auto-Poster

תוסף Chrome לפרסום אוטומטי בקבוצות פייסבוק.

## תכונות

- פרסום פוסט חדש בקבוצות מרובות ברצף אוטומטי
- פרסום תגובה בפוסט הראשון בכל קבוצה
- כותרת מובלטת בשורה נפרדת + גוף פוסט עם שמירת רווחים מדויקת
- צירוף קישור עם המתנה לטעינת תצוגה מקדימה לפני הפרסום
- השהיה אקראית 110–130 שניות בין קבוצה לקבוצה (נגד חסימות)
- לוג ריצה מפורט עם אפשרות העתקה
- ממשק RTL בעברית

## התקנה

1. הורד את הקבצים (או שכפל את הריפו)
2. פתח Chrome ועבור לכתובת `chrome://extensions`
3. הפעל **Developer mode** (פינה ימנית עליונה)
4. לחץ **Load unpacked** ובחר את תיקיית הפרויקט
5. התוסף יופיע בסרגל הכלים

## שימוש

1. לחץ על אייקון התוסף בסרגל הכלים
2. מלא:
   - **כותרת הפוסט** — תופיע בשורה ראשונה מובלטת
   - **גוף הפוסט** — רווחי שורה נשמרים כמות שהם
   - **קישור** (אופציונלי) — יופיע שתי שורות מתחת לתוכן
3. הוסף כתובות URL של קבוצות יעד
4. לחץ **התחל פרסום פוסט** או **התחל פרסום תגובה**
5. התוסף יפתח כל קבוצה בכרטיסייה, יפרסם, יסגור ויעבור לבאה

## קבצים

| קובץ | תפקיד |
|------|--------|
| `manifest.json` | הגדרות התוסף (Manifest V3) |
| `popup.html` | ממשק המשתמש |
| `popup.js` | לוגיקת UI ותקשורת עם Background |
| `background.js` | תור עבודה, פתיחת לשוניות, תזמון |
| `content.js` | הזרקת DOM לפייסבוק וביצוע הפרסום |

## דרישות

- Google Chrome (או Chromium)
- חשבון פייסבוק מחובר בדפדפן
- חברות בקבוצות היעד

## הערות

- התוסף פועל רק על `facebook.com`
- כל הנתונים נשמרים מקומית ב-`chrome.storage.local`
- אין שרת חיצוני — הכל רץ בדפדפן בלבד
